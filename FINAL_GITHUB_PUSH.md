## 🔐 Cuando pida contraseña:

**Username:** Tu usuario de GitHub  
**Password:** Token generado en https://github.com/settings/tokens

### Cómo generar token:
1. Ve a https://github.com/settings/tokens
2. Click "Generate new token"
3. Selecciona: `repo`, `write:packages`
4. Copia el token
5. Úsalo como contraseña en Git
